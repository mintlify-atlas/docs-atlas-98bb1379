Dufs is a lightweight, single-binary file server written in Rust. Share files instantly over HTTP with no configuration required — then add access control, HTTPS, WebDAV, and more when you need it.

## Key features

- **Static file serving** with directory listings and drag & drop uploads
- **Access control** — per-path read/write permissions with optional authentication
- **WebDAV** — mount as a network drive on Windows, macOS, and Linux
- **Resumable transfers** — HTTP range request support for large files
- **File search** — find files recursively across the served directory
- **HTTPS/TLS** — serve over TLS with your own certificate

## Get started

<Steps>
  <Step title="Install Dufs">
    ```bash
    cargo install dufs
    ```
    [Other installation methods →](/installation)
  </Step>
  <Step title="Serve your directory">
    ```bash
    dufs
    ```
    Open [http://localhost:5000](http://localhost:5000) in your browser.
  </Step>
  <Step title="Enable all operations">
    ```bash
    dufs -A
    ```
    Allows upload, delete, search, archive, and hash operations.
  </Step>
</Steps>
